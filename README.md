## Featured Projects

### Areya — Advanced Medical AI Agent
> Autonomous AI agent that conducts structured, doctor-like conversations with patients using LLM reasoning, RAG retrieval, and voice interfaces.

`LangChain` `Ollama` `Qdrant` `Flask` `PostgreSQL` `STT/TTS`

- Multi-step agentic reasoning with tool selection and medical safety checks
- Retrieval-Augmented Generation over patient history and medical knowledge
- Voice and text multimodal interaction with emergency escalation logic
- **Status:** Production-ready | **Domain:** Healthcare AI

---

### Clinical Automation Workflows (n8n)
> Production-grade AI-powered automation suite for clinical operations — appointment management, clinical note generation, and medical documentation.

`n8n` `LLM Chains` `PostgreSQL` `STT/TTS`

- Intelligent appointment booking, rescheduling, and conflict detection
- Speech-to-Text and Text-to-Text clinical documentation pipelines
- Strict data validation ensuring no fabricated medical data
- **Status:** Production-ready | **Domain:** Healthcare Automation

---

### Skin Lesion Segmentation & Monitoring
> Deep learning system for accurate skin lesion identification using U-Net and ResUNet architectures on the ISIC 2018 dataset.

`Python` `TensorFlow/Keras` `OpenCV` `Streamlit` `U-Net & ResUNet`

- Pixel-level semantic segmentation for early skin cancer screening
- Comparison of U-Net vs ResUNet architectures on 2,596 dermoscopic images
- Interactive Streamlit interface for clinical visualization
- **Domain:** Medical Image Analysis
